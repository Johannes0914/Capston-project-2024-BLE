Development of a Multi-Sensor Logger and a Gateway Using Bluetooth Low Energy (BLE)

Project Overview

The Multi-Sensor Logger and Gateway project aims to provide a scalable, low-power solution for real-time environmental monitoring. The system collects data such as temperature, humidity, ambient light, and more, transmitting it via BLE to a gateway for aggregation and visualization using MQTT and Grafana.
